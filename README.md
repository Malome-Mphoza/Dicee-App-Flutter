# dicee_flutter_app

A flutter Application that has two images that reactively change when pressed

#FlutterBrew

## Components used in the app

- Stateless and stateful widgets.
- Dart data types.
- Dart Functions.
- Row widget.
- Flatbutton widget.
- Expanded widget.
- Dart Math Library for randomizing the dice.
